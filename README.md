# This is a toolbox image

It contains Kubectl Helm Git etc. for daily business on the CLI and in CI
pipelines.

## How to use it

You can docker pull it from <https://hub.docker.com/r/syseleven/kubectl-helm>

## Contained software versions

* Kubectl 1.18.2
* Helm 3.2.1
  * Helm Diff latest
* Git latest
* helmfile v0.116.0
* s3cmd v2.0.2
