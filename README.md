# What is this image for

Kubectl Helm Git etc. Toolbox as Docker image.

## Where can I get the built image

You can find it on [DockerHub](https://hub.docker.com/r/syseleven/kubectl-helm)

## Which software versions does the image contain

* Kubectl 1.17.3
* Helm 3.1.2
* Git latest
* helmfile v0.102.0
* s3cmd v2.0.2
